## النسخة العربية


### 📝 عن المشروع
لعبة **سودوكو تفاعلية** تم بناؤها من الصفر باستخدام تقنيات الويب الأساسية. تركز اللعبة على تقديم تجربة مستخدم (UX) احترافية ومنطق برمجي (Logic) دقيق يمنع الأخطاء ويوفر بيئة ممتعة للحل.

### ✨ المميزات الرئيسية
*   **🎯 نظام التحقق الذكي:** فحص فوري للأخطاء في الصفوف، الأعمدة، والمربعات الصغيرة.
*   **🔦 الهايلايت التقاطعي (Crosshair):** تظليل تلقائي للصف والعمود عند اختيار مربع لمساعدتك على التركيز.
*   **✅ تمييز الإنجاز:** تلوين الأرقام المكتملة في كافة الخانات (9 خانات صحيحة) بلون أخضر مميز.
*   **💾 الحفظ التلقائي:** استخدام الـ `LocalStorage` لحفظ حالة اللعبة، مما يتيح لك إغلاق المتصفح والعودة لاحقاً.
*   **🖱️ تحكم سريع:** مسح الرقم المضاف بضغطة ثانية (Double Click action) لتسهيل التعديل.
*   **📱 تصميم متجاوب:** واجهة برمجية متوافقة تماماً مع أجهزة الموبايل والتابلت والكمبيوتر.

### 🛠️ التقنيات المستخدمة
*   **HTML5:** لبناء الهيكل الأساسي.
*   **CSS3:** للتنسيق واستخدام الـ `Flexbox` والـ `Media Queries`.
*   **JavaScript (ES6):** لبناء المنطق البرمجي والتعامل مع الـ `DOM`.

## English Version

### 📝 About The Project
An **interactive Sudoku game** built from scratch. The project focuses on providing a professional User Experience (UX) and robust programming logic to ensure rules are followed correctly.

### ✨ Key Features
*   **🎯 Smart Validation:** Instant error detection across rows, columns, and 3x3 sub-grids.
*   **🔦 Crosshair Highlighting:** Automatically highlights the row and column of the selected cell for better focus.
*   **✅ Completion Visuals:** Numbers correctly placed in all 9 positions are highlighted in green.
*   **💾 Auto-Save:** Powered by `LocalStorage`, allowing you to resume your game even after closing the browser.
*   **🖱️ Fast Controls:** Double-click logic to quickly erase numbers without needing extra buttons.
*   **📱 Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.

### 🛠️ Technologies Used
*   **HTML5**
*   **CSS3** (Flexbox & Media Queries)
*   **Vanilla JavaScript** (DOM Manipulation & State Management)
